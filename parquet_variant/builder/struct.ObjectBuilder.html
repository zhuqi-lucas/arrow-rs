Expand description

A builder for creating Variant::Object values.

See the examples on VariantBuilder for usage.

Fields§

§parent_state: ParentState<'a>§fields: IndexMap<u32, usize>§parent_value_offset_base: usize

The starting offset in the parent’s buffer where this object starts

§parent_metadata_offset_base: usize

The starting offset in the parent’s metadata buffer where this object starts used to truncate the written fields in drop if the current object has not been finished

§has_been_finished: bool

Whether the object has been finished, the written content of the current object will be truncated in drop if has_been_finished is false

§validate_unique_fields: bool§duplicate_fields: HashSet<u32>

Set of duplicate fields to report for errors

When this is enabled, calling ObjectBuilder::finish will return an error if any duplicate field keys were added using ObjectBuilder::insert.

impl Drop for ObjectBuilder<'_>

Drop implementation for ObjectBuilder does nothing as the finish method must be called to finalize the object. This is to ensure that the object is always finalized before its parent builder is finalized.
